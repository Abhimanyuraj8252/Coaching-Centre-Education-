# TopCoachingInstitute — Website Template

This repository contains a static website template for a coaching/education centre.

Contents
- `index.html` — main page
- `privacy.html`, `terms.html` — policy pages
- `css/` — styles (contains `style.css`)
- `js/` — JavaScript (`script.js`)
- `images/` — assets used by the template

Quick start
1. Open `index.html` in your browser to preview the site.
2. Edit HTML, CSS, and images in-place to customise the template.

To contribute
- Create a branch for your changes: `git checkout -b my-feature`
- Commit and push, then open a pull request.
